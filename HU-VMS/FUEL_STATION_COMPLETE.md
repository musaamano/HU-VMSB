## 🚀 Usage Instructions

### Dispensing Fuel
1. Select vehicle from dropdown
2. Select driver from dropdown
3. Choose fuel type (Diesel/Petrol)
4. Enter liters amount
5. Enter odometer reading
6. Click "Verify Authorization"
7. Enter valid authorization code
8. Add optional notes
9. Click "Dispense Fuel"

### Generating Reports
1. Click "Generate Report" button
2. Select report type (Daily/Weekly)
3. Choose recipient (Admin/Transport Office)
4. Review report summary
5. Click "Generate PDF Report"
6. PDF downloads automatically

### Updating Inventory
1. Navigate to Fuel Inventory page
2. Click "Update Stock" on desired fuel type
3. Enter liters to add
4. Select reason
5. Click "Update Stock"
6. Inventory updates in real-time
